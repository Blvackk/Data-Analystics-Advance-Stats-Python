SQL JOINS
(INNER) JOIN : Returns records that have matching values in both tables
LEFT (OUTER) JOIN : Returns all records from the left table, and the matched records from the right table
RIGHT (OUTER) JOIN: Returns all records from the right table , and the matched records from the left table
FULL(OUTER) JOIN: Returns all records when there is a match in either left or right table

When we works with SQL JOINS it always works with minimum 2 table

CROSS JOIN : combines every row from the first table with every row from the second table, returning the Cartesian product. If Table A has m rows and Table B has n rows, the result will have m × n rows.

Assymetry ---->

Skewness indicates whether the data is concentrated  on which side , either left side or right side

SKEWNESS is divided into 3 :-
Positive skewness ---> MEAN > MEDIAN & MODE  ( Data is left & Outlier is at right )
Negative Skewness ---> MODE> MEDIAN > MEAN  ( Data is right & Outlier is  at left
Zero Skewness ---> MEAN = MEDIAN = MODE ( Data is at centre & no Outlier is present ) 

ZERO SKEWNESS == NORMAL DISTRIBUTION == GAUSSIAN DISTRIBUTION == BELL CURVE == 0 SYMMETRY


SKEWNESS == Graphs are Flatten
KURTOIS == Graphs are tailed

+VE skew  =  +VE Kurtois ( Leptokurtic)
-VE Skew = -VE Kurtois ( Playtykurtic)
0 Skew = Normal Distribution (Meso Kurtic)

Linear regression line -- 
y = mx+ c        ; y = Actual DATA
y^ = mx + c      ; y^ = Predicted DATA


x - iv 
y - dv
m - slope
c - constant | intercept
y - actual data
y^ - predicted data


Best Fit Line - the line has least fit error is called best fit line
ERROR --> Distance between actual point & Predicted point is called ERROR


ABSOLUTE ERROR :- (y-y^)
SQUARE ERROE :- (y-y^)**2
ROOT MEAN SQUARE ERROR :- np.sqrt(y-y^)



ANOVA -- Analysis of Variance
  ML is divided into 3 TYPES :-
     1. REGRESSION
     2. CLASSIFICATION
     3. CLUSTERING



Performance Measure for Regressions --> R2 & Adjusted R2
If r2 & adjusted r2 between range of 0-1 then regression model is best model



SSE (SUM OF SQUARE ERROR) :- ACTUAL POINT - PREDICTED POINT
SSR (SUM OF SQUARES REGRESSION) :-  PREDICTED POINT - AVERAGE POINT
SST = SSE + SSR => Actual - average = (actual - prediction ) + ( pred - avg)


R2 = 1 - SSR/ SST

If you build regression Model with more attribute then adjusted R2 will come to picture.
performance measure of regression model :-
 r2 , adjusted r2
 range of 0 - 1 | r2 >= adjusted r2  OR adjusted R2 < r2

  1 dv & many iv
  R2 & ADJUSTED R2
