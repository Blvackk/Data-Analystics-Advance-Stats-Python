(24-03-26)

SET OPERATIONS ==
Union  --> Combines elements from both sets, removing duplicate. set1 | set 2 OR set1.union(set2)

Intersection  --> Keeps only the elements common to both sets. set1 & set2  OR  set1.intersection(set2)

Difference --> Keeps elements present in the first set but not in the second set. set1 - set2  OR  set1.difference(set2) 

Symmetric difference --> Keeps elements present in either set, but not in both. set1 ^ set2 OR  set1.symmetric_difference(set2)


symmetric difference update --> Modifies the original set in-place to keep elements unique to each set.  set1 ^= set2  OR set1.symmetric_difference_update(set2)

intersection_update --> Modifies the original set in-place to keep only common elements.
set1 &= set2  OR set1.intersection_update(set2)


superset --> Checks if a set contains all elements of another set, Returns a boolean. set1 >= set2 OR  set1.issuperset(set2)


subset-->  Checks if all elements of a set are contained within another set. Returns a Boolean. set1 <= set2  OR  set1.issubset(set2)


disjoint --> Checks if two sets have zero elements in common. Returns a boolean.
set1.isdisjoint(set2)
